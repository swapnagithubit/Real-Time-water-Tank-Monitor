# Automatic tank water prediction

This repository contains a real-time water tank monitoring frontend and backend.

Structure

- `backend/`: Node.js Express + Socket.io server that simulates sensor data
- `frontend/`: React app that visualizes the tank, chart, alerts, and reports

Quick start (development)

1. Start backend

```bash
cd backend
npm install
node server.js
```

2. Start frontend

```bash
cd frontend
npm install
npm start
```
